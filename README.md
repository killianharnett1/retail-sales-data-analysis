## Retail Sales Data Analysis

This project focuses on analyzing retail store sales data to uncover patterns, trends and key factors that influence transaction value. 

The analysis includes data cleaning, exploratory data analysis (EDA) and statistical techniques to better understand customer purchasing behavior and sales performance.

### Key Features
- Data cleaning and preprocessing using Python
- Exploratory Data Analysis (EDA)
- Statistical analysis of transaction values
- Identification of key factors affecting retail sales

### 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

### Dataset
The dataset contains retail transaction records used to analyze sales trends and customer behavior.

### Outcome
The project provides insights into the variables that significantly impact transaction value helping inform data-driven decision-making in a retail context.
